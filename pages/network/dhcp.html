DHCP

Overview

  • Scope: valid IP addresses DHCP can assign
  • Reservations: certain IPs go to certain devices
  • Lease: default is 24 hours, corporate is 7-30 days
  • Needed configurations: IP address, subnet mask, default gateway, DNS server IP
  • APIPA: when DHCP fails, assigns static address
