Key Exchange Algorithms

Diffie-Hellman

Note: g is primitive mod n if for every integer a coprime to n there is some integer k for which gk ≡ a (mod n). Two numbers are coprime if they have no common factors other than 1. n should be large and (n - 1)/2 should be prime

  1. Alice and Bob agree on a large prime, n and g such that g is primitive mod n
  2. Alice chooses a random large number a and sends Bob A = ga mod n
  3. Bob chooses a random large number b and sends Alice B = gb mod n
  4. Alice computes k = Ba mod n
  5. Bob computes k = Ab mod n
  6. They now have a shared key k
